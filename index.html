Lecture des indicateurs

Une saisie « journée complète » vaut 1 journée équivalente. Une saisie du matin ou de l’après-midi vaut 0,5 journée. Le taux de couverture compare les demi-journées renseignées aux demi-journées ouvrées théoriquement disponibles pour les agents actifs du périmètre. Il mesure la complétude du planning, pas un taux d’absentéisme réglementaire.

Administration

Contrôle technique, maintenance et accès aux informations du widget — réservé aux administrateurs
Session administrateur active. Cet onglet est masqué dès la déconnexion. La protection par mot de passe concerne l’interface du widget ; les autorisations réelles sur les données doivent également être configurées dans Grist.

Outils de maintenance

Recharger les données

Relit les tables Grist et actualise toutes les pages.

Période collective

Ajoute un même code sur une période pour l’ensemble des agents actifs, notamment lors des congés obligatoires.

Table du personnel

Modifie toutes les informations de T01_Personnel, y compris les rubriques RH habituellement en lecture seule.

Suppression définitive

Supprime un agent ainsi que ses présences et son horaire associés, après double confirmation.

Exporter un diagnostic

Télécharge un CSV avec les volumes, tables et informations techniques.

Mode application

Ferme la session administrateur et réactive l’affichage applicatif.

Interface Grist

Quitte le plein écran afin de retrouver les menus et tables du document.

Configuration de sécurité

Le mot de passe administrateur actuel est défini dans le fichier HTML par la constante ADMIN_PIN.

Pour modifier le mot de passe, recherchez const ADMIN_PIN='SELACV4' dans le fichier et remplacez uniquement la valeur entre apostrophes.

Modifier les horaires

La quotité et le nombre de jours travaillés sont enregistrés directement dans T10_Horaires. Le contrôle applique la cible exacte correspondant à la quotité. Pour une quotité de 100 %, la cible est 35 h 15, ou 34 h 15 si une plage commence à 06:00. Les pauses entre matin et après-midi ne sont pas comptées.
